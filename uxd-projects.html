Northern Fireworks

This program was created at the programming bootcamp of the UXD course at The Hague University of Applied Sciences. It was a free assignment. The idea is a Northern Firework above the skyline of The Hague.


If the user presses the mouse button on a particular place on the screen, a rocket will be shot from the bottom to this pressed point.


Product characteristics:

  • Professional Usage of the programming language "Processing"
  • Aesthetically pleasing fireworks, which the user is going to watch the rest of the day

Game "Flying Ellipse"

This is a deeply engaging desktop computer game, in which the user needs to survive a moving corridor. The corridor moves from the right to the left and the flying ellipse needs to be moved up and down by using the arrow keys. When the user gets further in the corridor, the game gets harder and harder.


Product characteristics:

  • The game is programmed with the modern programming language "processing"
  • Coding includes the usage of two and three dimensional arrays
  • High performance of the code is approved by experienced programmers from The Hague University of Applied Sciences
  • Different complex mathematical functions, classes and selfmade objects are used
